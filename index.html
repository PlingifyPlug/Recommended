Editing Software:

1. Davinci Resolve-There is a link to this software in our URL Box on the homepage! This is a free editing software that many big film companys use due to its huge variety of sound effects, text options, editing options & effects aswell as its best color grading! This is a free software or a one time payment of £239.99 for there studio option. This is the sofware we use for out Tiktok and YT!

2.Premier Pro-There is no lnk in the url search box we have but if you search adobe it will come up on there page the only issue is that you have to pay to use this equiptment. £30 a month or £239.99 a year. This has really good affects and a good ui and is easy to learn once started to use it but it requiers 'after affects(read below!)' as Premier Pro is the main base then after affects has the effects for after. after affects is in the same package you pay for with Premier Pro. You dont get as good color grading but it is still really good but the price isnt the best so i would stick with Davinci Resolve!
